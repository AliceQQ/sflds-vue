# 头部组件
头部组件是 [sf-view](./view.html) 组件的子组件。

## 示例代码

```html
<sf-header>...</sf-header>
```

## 效果图

![preview](./media/header.png)

## API
以下是 `<sf-header>` 组件可用的 props：

| 属性 | 类型 | 默认值 | 说明 |
| :---: | :---: | :---: | :--- |
| website-cn | String | `证券期货法规数据库系统` | 站点中文名 |
| website-en | String | `Securities and futures law database` | 站点英文名 |
